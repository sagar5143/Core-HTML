HTML Tags

HTML Tags

  1. <html>: The <html> tag represents the root of an HTML document. All other HTML elements must be descendants of this element.
  2. <head>: The <head> tag contains meta-information about the HTML document, such as its title and links to stylesheets.
  3. <title>: The <title> tag sets the title of the HTML document, which is displayed in the browser's title bar or tab.
  4. <body>: The <body> tag represents the main content of the HTML document. It contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  5. <h1> to <h6>: The <h1> to <h6> tags are used to define HTML headings. <h1> defines the most important heading, and <h6> defines the least important heading.
  6. <p>: The <p> tag defines a paragraph of text. It automatically adds some space before and after the paragraph.
  7. <a>: The <a> tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the <a> element is the href attribute, which indicates the link's destination.
  8. <img>: The <img> tag is used to embed images in an HTML document. It has two main attributes: src (source) and alt (alternative text).
  9. <ul> and <ol>: The <ul> (unordered list) and <ol> (ordered list) tags are used to define lists. <ul> creates a bulleted list, and <ol> creates a numbered list.
  10. <li>: The <li> tag defines a list item and must be used inside <ul>, <ol>, or <menu> tags.
  11. <div>: The <div> tag is a block-level container used to group other elements together. It is useful for applying styles and layout to sections of a document.
  12. <span>: The <span> tag is an inline container used to group text or other inline elements. It is useful for applying styles to parts of a text.
